automoderator
=============

## Setup

 1. Create a Reddit app here - https://www.reddit.com/prefs/apps.
 1. Clone submodules.
 2. ```sh
    cd reddit-oauth-helper/
    yarn
    node node-server.js
    ```


## Subreddit setup.

 1. Configure link flair position to "left" in order to display flairs.
